== Description ==

Simple free press release distribution and marketing tool. Submit unlimited free press releases to PR Wire Pro's huge press release database and social press release distribution network. All free submissions include social marketing with submission to several top social networks, press release distribution sites, article directories, and rss feeds on the web. Our free press release submission tool builds high pr links to your press release that stick to increase your google page rank and seo for your website. Visit our website http://prwirepro.com to view our other marketing and submission services.

== Installation ==

This section describes how to install the plugin and get it working.

1. Install through your wordpress plugin panel or Upload `prwirepro.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open the Pr Wire Pro dashboard from within your wordpress panel.
4. Fill our the form and submit unlimited press releases to our network.
5. Donate if you like our plugin http://prwirepro.com/donate/
